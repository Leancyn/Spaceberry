Module 5: Related Time Expressions

A. Objectives

  • Identify expressions related to time
  • Understand temporal relationships using before, after, during, since, etc.

B. Material Description

This module focuses on time-related expressions that help understand the sequence of events in conversations. Common expressions include:

  • Before the meeting
  • After the break
  • During the lecture
  • Since yesterday

These expressions help define when an event happens and are often used to answer questions accurately in listening exercises.

C. Review Exercises

Identify the correct sequence of events and match time expressions to the appropriate context in conversations.
